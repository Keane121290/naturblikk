# Streaming

Denne mappen dokumenterer hele videokjeden i Naturblikk.

## Planlagt arkitektur

Tapo C310
↓
RTSP
↓
OBS Studio
↓
MediaMTX
↓
Naturblikk.no

## Maskinvare

- 2 × TP-Link Tapo C310

## Programvare

- OBS Studio
- MediaMTX
- AI-verktøy (senere)

## Fremtidige funksjoner

- AI-basert fuglegjenkjenning
- AI-lydfiltrering
- Automatisk høydepunkter
- Flere kameraer

## Viktige krav

- Stabil 24/7 drift
- Lav forsinkelse
- Enkel feilsøking
- Backup av OBS-profiler
