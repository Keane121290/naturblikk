# 🎨 Naturblikk – Mockups

Denne mappen inneholder designutkast og konsepter for Naturblikk.

## Formål

Mockupene fungerer som visuelle mål for utviklingen. WordPress og senere frontend skal bygges med utgangspunkt i disse designene.

## Første konsept

**Status:** Godkjent som designretning.

Designet legger vekt på:

- Stor og rolig hero-seksjon
- Livestream som hovedfokus
- Naturlige grønn- og jordtoner
- Enkel navigasjon
- Mobilvennlig oppsett
- Skandinavisk, minimalistisk uttrykk

## Prinsipper

- Livestreamen er prosjektets viktigste innhold.
- Store bilder og mye luft.
- Ingen unødvendige elementer eller visuell støy.
- Design skal fungere like godt på mobil som på desktop.

## Neste steg

- Legge inn første mockup (homepage-v1)
- Lage mockup av Live-siden
- Lage mobilversjon
- Tilpasse designet til WordPress/Kadence
